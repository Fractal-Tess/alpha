# Tech Stack

| Component          | Choice                                  |
| ------------------ | --------------------------------------- |
| **Frontend**       | SvelteKit + shadcn-svelte + Tailwind v4 |
| **Backend**        | Convex Cloud + Workpools + Workflows    |

| **Doc Processing** | Docling (Python HTTP microservice)      |
| **AI Gateway**     | OpenRouter                              |
| **Default Model**  | DeepSeek v3.2                           |
| **Monorepo**       | Turborepo + Bun                         |
| **Testing**        | Vitest (unit) + Playwright (e2e)        |
| **Deployment**     | Dokploy + Docker Compose (self-hosted)  |
