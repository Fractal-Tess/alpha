# Key Features

## Phase 1: MVP (Core Study Tools)

### 1. Flashcards

- AI-generated from document content
- SM-2 spaced repetition scheduling
- Flip card UI with keyboard shortcuts

### 2. Quizzes

- Multiple choice questions
- AI-generated explanations for wrong answers
- Progress tracking

### 3. Summaries

- Section-by-section summaries
- Key concepts extraction
- Hierarchical structure preservation

### 4. Sharing

- Generate share codes for decks
- Optional public sharing
- Import shared decks

## Phase 3: Advanced Features

### 8. Study Guide Generator

- AI creates structured study plan from documents
- Topics organized by importance/difficulty
- Time estimates per section
- Pre-test and post-test assessments
- Customizable study schedule

### 9. Audio Recording & Lecture Notes

- Record audio in-app or upload recordings
- Transcription via Whisper API
- Auto-generate structured notes
- Highlight key points and action items
- Sync with document content

### 10. Practice Problem Generator

- AI generates similar problems for practice
- Adjustable difficulty levels
- Step-by-step solutions
- Great for STEM subjects
- Tracks mastery per topic

### 11. Concept Map

- Visual graph of topic relationships
- Interactive node exploration
- Auto-generated from document structure
- Manual editing support
- Export as image

### 12. Pomodoro Timer

- Built-in focus timer (25/5 or custom)
- Integrates with study sessions
- Break reminders
- Session statistics

### 13. Export Options

- Export flashcards to Anki (.apkg)
- Notes to Markdown/PDF
- Quizzes to printable format
- Study guides to PDF
